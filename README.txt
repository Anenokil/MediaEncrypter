Программа осуществляет шифровку/дешифровку изображений, гифок и видео

Поддерживаемые расширения файлов: .png .jpg .jpeg .bmp .gif .avi .mp4 .webm
   Изображения, полученные в результате работы программы сохраняются с расширением .png
   Гифки,       полученные в результате работы программы сохраняются с расширением .gif
   Видео,       полученные в результате работы программы сохраняются с расширением .mp4
Сейчас в папке с входными файлами находятся тестовые файлы. На них можно протестировать работу программы. При ненадобности их все можно удалять

В программе есть изменяемые настройки (чтобы в них зайти, запустите программу и введите S)

По умолчанию при   шифровке файлы берутся из папки f_src и сохраняются в папку f_enc
По умолчанию при дешифровке файлы берутся из папки f_enc и сохраняются в папку f_dec

Перед осуществлением шифровки/дешифровки вам будет необходимо ввести ключ
   Ключ может содержать только цифры, латинские буквы, - и _
   Этот ключ вы придумываете сами
   Ключ должен иметь строго фиксированную длинну (она будет указана при вводе ключа в программе)
   Также вы можете протестировать программу с тестовым ключём (он будет предложен при вводе)
САМОЕ ВАЖНОЕ: чтобы дешифровать файл, вы должны ввести тот же ключ, который вводили при шифровке
